Find!

Step 1: define this page as your default search engine.

Step 2: type your search in the URL bar.

You can use the following syntax to prefix your search term, and search directly on the corresponding site:

  • !a = amazon.com
  • !c = contacts.google.com
  • !ciu = caniuse.com
  • !d = duckduckgo.com
  • !dd = devdocs.io
  • !dr = drive.google.com
  • !g = encrypted.google.com
  • !lp = lpepfinder.com
  • !m = maps.google.com
  • !osm = openstreetmap.org
  • !w = en.wikipedia.org
  • !y = youtube.com

Learn more and audit the code on Github.

One last thing, the keyboard shortcut to move your cursor directly inside your browser URL bar is Ctrl + l (all browsers). Try it!
