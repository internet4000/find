Find!

Step 1: make Find! your default search engine.

Step 2: type your search in the URL bar.

Use the following !keywords before your query to go directly on the corresponding site.

  • !a = amazon.com
  • !c = contacts.google.com
  • !ciu = caniuse.com
  • !d = duckduckgo.com
  • !dd = devdocs.io
  • !dr = drive.google.com
  • !e = rover.ebay.com
  • !g = encrypted.google.com
  • !k = keep.google.com
  • !l = linguee.com
  • !lbc = leboncoin.fr
  • !lp = lpepfinder.com
  • !m = maps.google.com
  • !osm = openstreetmap.org
  • !tr = translate.google.com
  • !w = en.wikipedia.org
  • !wa = wolframalpha.com
  • !y = youtube.com

The Ctrl-l keyboard shortcut, on any page and in any browser, will focus the URL bar.
